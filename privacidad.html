Política de Privacidad

Última actualización: 19 de julio de 2026.

Bingo Ganga usa la información suministrada por los participantes únicamente para gestionar compras de cartones, verificar pagos, administrar jugadas, publicar ganadores y contactar a los usuarios cuando sea necesario para el desarrollo del juego.

Datos que podemos solicitar

Podemos solicitar nombre y apellido, teléfono, cédula, cédula de referido cuando aplique, cartones seleccionados, referencia bancaria, monto pagado, datos para entregar premios y comprobante de pago. Estos datos son necesarios para confirmar la inscripción y evitar duplicidad o errores.

Comprobantes y archivos

Los comprobantes se almacenan en un espacio privado. Solo las cuentas administradoras autenticadas pueden revisarlos para validar pagos, aprobar o rechazar inscripciones y resolver reclamos.

Uso y conservación

La información se conserva durante el tiempo necesario para operar el bingo, auditar pagos, resolver dudas y mantener el historial de ganadores. Cuando ya no sea necesaria, podrá ser eliminada o depurada por administración.

Seguridad

El sistema usa Supabase con control de acceso, autenticación administrativa y reglas de seguridad en la base de datos. Limitamos el acceso a los datos personales al personal autorizado.

Publicación y compartición

No vendemos información personal. Podemos publicar datos mínimos de ganadores o aprobados, como nombre, cartón y cédula parcialmente ocultada, cuando sea necesario para la transparencia del juego.

Tus derechos

Puedes solicitar acceso, corrección o eliminación de tus datos escribiendo a Bingoganga012@gmail.com. Es posible que conservemos datos indispensables si existe una operación, pago, reclamo o auditoría pendiente.

Contacto

Para preguntas sobre privacidad o uso de datos, escribe a Bingoganga012@gmail.com.
